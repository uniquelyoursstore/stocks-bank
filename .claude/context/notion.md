## סכמת שדות — בנק המניות

| שדה בנוטיין | טיפוס | ערכים אפשריים |
|------------|-------|--------------|
| שם החברה | title | טקסט חופשי |
| טיקר | rich_text | אותיות גדולות בלבד |
| סקטור | select | חלל / הגנה / אנרגיה / בריאות / שבבים / פיננסים / מטריאלס |
| שכבה | select | ליבה / צמיחה / ספקולטיבי / המתנה |
| סיכון | select | נמוך / בינוני / גבוה |
| מחיר נוכחי | number | מספר בלבד, ללא $ |
| מחיר יעד | number | מספר בלבד, ללא $ |
| משקל מוצע % | number | מספר בלבד, ללא % |
| תזה | rich_text | טקסט חופשי |
| פונדמנטל | rich_text | טקסט חופשי |
| טכני | rich_text | טקסט חופשי |
| נוסף על ידי | select | אייל / גיא / רוני |
| סטטוס | select | פעיל / בבדיקה / הוסר |

**כלל חשוב:** שכבה "המתנה" → סטטוס "בבדיקה". כל שאר השכבות → סטטוס "פעיל".

## מיפויים לקוד

```js
// סקטורים עברית → קוד
sectors: {
  אנרגיה: 'energy',
  חלל: 'space',
  הגנה: 'defense',
  בריאות: 'health',
  שבבים: 'chips',
  פיננסים: 'financials',
  מטריאלס: 'materials',
}

// שכבות עברית → קוד
layers: {
  ליבה: 'core',
  צמיחה: 'growth',
  ספקולטיבי: 'speculative',
  המתנה: 'waiting',
}

// סיכון עברית → קוד
risk: {
  נמוך: 'low',
  בינוני: 'medium',
  גבוה: 'high',
}
```

## פורמט STOCK_ADD

כשמתקבל הפורמט הזה — מוסיפים מיד לנוטיין, ללא שאלות:

```
STOCK_ADD | טיקר: X | שם: Y | סקטור: Z | שכבה: W | סיכון: V | מחיר נוכחי: $N | מחיר יעד: $M | משקל מוצע: P% | תזה: ... | פונדמנטל: ... | טכני: ...
```
