# Features — טאבים, משימות פתוחות, סקילים

## 8 הטאבים הקיימים

| # | טאב | מה הוא עושה |
|---|-----|------------|
| 1 | בנק | רשימת כל המניות מנוטיין + מחיר חי מ-Yahoo + כפתור חדשות לכל מנייה |
| 2 | תיקים | P&L ו-Win Rate לפי תיק (אייל / גיא / רוני) |
| 3 | טריידים | היסטוריית עסקאות עם פילטר פתוח / סגור / הכל |
| 4 | היט-מאפ | ויזואליזציה לפי סקטור, צבע לפי P&L % |
| 5 | מחשבון R:R | קלט: כניסה + סטופ + יעד + הון + % סיכון → מספר מניות + גודל פוזיציה |
| 6 | יומן דוחות | Earnings קרובים ועברו, ממוין לפי תאריך |
| 7 | השוואה | השוואה בין 2 מניות מהבנק לפי כל המדדים |
| 8 | עדכון יומי | פוזיציות פתוחות + P&L לא ממומש |
