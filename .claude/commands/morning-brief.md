## פונקציה
בריף בוקר יומי: מחירים עדכניים + חדשות + עדכון דאשבורד לכל המניות בבנק.

## פרומפט

## מטרה
תמונת מצב יומית מלאה — מניות + ניהול פרויקט. בסוף: דאשבורד מעודכן נשלח אוטומטית.

## טריגרים
- "בריף" / "בוקר טוב" / "מה קורה היום" / "עדכון בוקר"

## תהליך ביצוע — 5 שלבים

### שלב 1: קרא קבצי זיכרון
קרא את /mnt/project/project-status.md ו-/mnt/project/project-stack.md — תצטרך אותם לסקשן המשימות.

### שלב 2: חיפוש מחירים (מקסימום 4 חיפושים)
1. RKLB ASTS PL BKSY RDW UFO LUNR IRDM stock price today
2. ITA PLTR KTOS AVAV LHX LMT CACI RCAT ESLT stock price today
3. VST CEG CCJ NLR TAN stock price today
4. LLY JNJ UNH MRNA NVDA AVGO MU INTC ON CEVA MTSI ONTO stock price today

### שלב 3: חיפוש חדשות (ממוקד)
רק: מניות שזזו >3% + דוחות בפחות מ-14 יום + מניות המתנה עם קטליסט קרוב.

### שלב 4: ניתוח ודירוג
- דחוף: שינוי >5%, תזה שהשתנתה, קטליסט שהתגשם, טריגר המתנה
- חשוב: שינוי 2-5%, דוח קרוב, אנליסט שינה יעד
- מידע: שינוי <2%, עדכון שוטף

### שלב 5: עדכון דאשבורד — שלושה חלקים

#### 5א — עדכון מחירים
בקובץ /mnt/user-data/outputs/stock-bank.jsx — עדכן currentPrice לכל מנייה שיש לה מחיר חדש.

#### 5ב — עדכון סקשן עדכון יומי
החלף את SEED_DAILY_UPDATES בקובץ הדאשבורד עם רשומות היום:

const SEED_DAILY_UPDATES = [
  {
    id: "brief-DDMMYY-N",
    date: "YYYY-MM-DD",
    type: "מחיר|חדשה|אזהרה|ניתוח|קטליסט",
    ticker: "TICKER",
    title: "כותרת קצרה",
    content: "תיאור 2-3 שורות",
    impact: "חיובי|שלילי|ניטרלי|צריך מעקב",
    priority: "דחוף|חשוב|מידע",
    change: X.XX,
    price: XXX.XX,
    action: "פעולה מוצעת"
  },
  // רשומת PROJECT_STATUS בסוף — חובה:
  {
    id: "brief-DDMMYY-project",
    date: "YYYY-MM-DD",
    type: "ניהול",
    ticker: "PROJ",
    title: "משימות פתוחות",
    content: "[משימה קריטית 1] | [משימה קריטית 2] | [משימה חשובה]",
    impact: "צריך מעקב",
    priority: "חשוב",
    change: null,
    price: null,
    action: "ראה סקשן משימות בבריף"
  }
];

#### 5ג — שלח את הדאשבורד
אחרי שמירת הקובץ — שלח אותו למשתמש עם present_files.
