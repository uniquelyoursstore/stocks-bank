# alert-add — הוספת התראת מחיר

> העתק לכאן את הפרומפט המלא מהסקיל בצ'אט הרגיל.

## פונקציה
הוספת התראת מחיר לטיקר → עדכון `public/alerts.json` בגיטהאב.

## פורמט alerts.json
```json
{
  "alerts": [
    { "ticker": "TICKER", "direction": "above|below", "triggerPrice": 0.00 }
  ]
}
```
