# update-memory — עדכון קבצי הזיכרון בסוף שיחה

עבור על השיחה הנוכחית מהתחלה ועדכן את קבצי הזיכרון לפי מה שהשתנה.

---

## מה לבדוק בכל קובץ

### `.claude/context/stocks.md`
- נוספה מנייה חדשה? → הוסף לטבלה המתאימה לפי סקטור
- שכבה של מנייה השתנתה? → עדכן
- טריגר הושג או השתנה? → עדכן / הסר מרשימת ההמתנה
- עדכון מחיר או חדשה חשובה? → הוסף ל"עדכונים אחרונים"

### `.claude/context/notion.md`
- נוסף שדה חדש לנוטיין? → הוסף לסכמה
- השתנה DATABASE_ID? → עדכן
- נוסף פורמט פקודה חדש? → הוסף

### `.claude/context/features.md`
- נוספה פיצ'ר או טאב חדש? → עדכן את הרשימה
- משימה הושלמה? → סמן ✅ ועבור לחלק "הושלם"
- נוספה משימה חדשה? → הוסף בעדיפות המתאימה
- נוסף סקיל חדש? → הוסף לטבלה

### `.claude/context/team.md`
- השתנה משהו בצוות, כלים, תהליכים או כללי אבטחה? → עדכן

---

## כללים
- עדכן **רק מה שבאמת השתנה** בשיחה הזו — אל תכתוב מחדש מה שכבר נכון
- אל תמחק מידע קיים אלא אם הוא מיושן במפורש
- אם לא השתנה כלום בקובץ מסוים — אל תיגע בו
- לאחר כל העדכונים: `git add .claude/ && git commit -m "update memory: [תיאור קצר]" && git push origin main`
